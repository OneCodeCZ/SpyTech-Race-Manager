Racers

The Race Manager software allows you to keep the database of your racers. For each racer, the following items may be stored:

  • Name
  • Surname
  • Display name
  • Identification number
  • Club name
  • Licence number

A new racer may be written in by the New racer command in menu File | New, and also in context menu of the folder of Racers within the Database tree on the left margin of the screen. You may invoke this function also pressing the F4 key or clicking on the icon. The meaning of all items corresponds to their captions. The Display name is by default set to the value of Surname but may be modified later. Some racers have their favorite nicknames and they insist on using them.
