Program guide

The Race Manager software serves the racers the following functions:

  • Training
  • Race measurement

On this page you find a brief description of both of them.

How to manage a training

Step 1: Software and sensor setup

If you run the software for the first time on your computer, it is probable that the initial settings has not yet been performed. In this step you put in your track data and set the STM/2008 device so that it correctly reads the track outputs. The dialogue window for the setup is in menu Tools, item Settings. You may find more information on the page Setup.

Step 2: The training mode start

You may run the training mode using the command Tools | Training. The dialog window which pops up allows you to choose the training type. More info on the page Training.

How to manage a race

Step 1: Software and sensor setup

See step 1 of the previous paragraph.

Step 2: Database of racers.

Before the race may start, it is necessary to specify which racers are about to attend the race. In the race definition phase (step 4), the racers are chosen from the database of racers which has to be filled up with relevant data before the race defition starts. These data are persistent and are available every time you run the software. See page Racers for details.

Step 3: Race systems

The race measurement is driven by rules specified by the race system. For instance, the duration of rides, the lane exchange rules, the count of the racers to be advanced into subsequent race parts etc. The race systems are stored and used in a similar manner as the racers. See page Race systems for details.

Step 4: Race definition

The race is defined by choosing a race system and racers, both already defined earlier. See the page Race definition.

Step 5: Race measurement

The qualification phase is measured according to different rules than the measurement of the other parts of the race. See qualification measurement and measurement of the other race parts respectively. Various data and score sheets are generated as the race is in progress. These sheets may be viewed, printed and exported into HTML. See Score sheets.
